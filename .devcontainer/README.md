# Devcontainers
[Documentación oficial](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)
## Requisitos
- Haber ejecutado el /.windows/setup.ps1
- Haber instalado las extensiones recomendadas de /.vscode/extensions.json
> Podéis poner @recommended en el buscador de extensiones de VSCode y ahí os aparecerán.

Básicamente esta extensión de VSCode nos permite tener exactamente el mismo entorno de desarrollo a todos.

![TEST](https://microsoft.github.io/vscode-remote-release/images/remote-containers-readme.gif)

Cada vez que habras el workspace, abajo a la derecha te aparecerá para crear el devcontainer y automáticamente conectar tu ventana de VSCode a él.
